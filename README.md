# Microsoft Rewards Macro

Welcome to the Microsoft Rewards Macro! This project allows you to automatically gain Microsoft Rewards points by using a simple system to open a new Bing tab, click the search bar, type a new prompt, and repeat the process! Although this sounds cool, you should be mindful when using it, as it could get your account banned, so don't use it as the only way to get points.

To start it, download the latest release, and run the 'run.bat' file. You can stop it at any time by opening Command Prompt and pressing Ctrl+C.

# *This so far only works on Windows*
